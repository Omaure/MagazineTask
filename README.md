# For Trying
- Node : 
    - open Magazine Server
    - npm install 
    - nodemon index.js
- testing Nodejs (unit test) :
    - open Magazine Server
    - npm install
    - npm run test
- React :
    - open magazinefrontend folder
    - npm install
    - npm start

# Features
- Login
- Sign Up
- Logout
- View Articles
- Post an Article
- Edit an Article
- Delete an Article
